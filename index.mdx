## Security & Enterprise

Cardinal provides enterprise-grade security and compliance features:

**Data Protection**
- End-to-end encryption for all document uploads and API communications
- Zero data retention policy - your documents are processed and immediately deleted
- SOC 2 Type II compliance with regular third-party security audits

**Access Control**
- API key authentication with granular permissions and rate limiting
- Role-based access controls for team management and document sharing
- Audit logs for all processing activities and API access

**Infrastructure**
- AWS-hosted infrastructure with 99.9% uptime SLA
- Auto-scaling processing clusters for consistent performance
- Global CDN for fast document uploads from anywhere
