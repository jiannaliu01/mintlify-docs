## Prerequisites
- [Install Docker](https://docs.docker.com/get-docker/)  
- [Install AWS CLI](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) with access to AWS ECR  

---

## Authenticate to ECR
Cardinal images are hosted in a private ECR. Contact **team@trycardinal.ai** for access.  

```bash
export AWS_ACCESS_KEY_ID=your_access_key_id
export AWS_SECRET_ACCESS_KEY=your_secret_access_key

aws ecr get-login-password --region us-east-2 \
  | docker login --username AWS \
  --password-stdin <your-cardinal-ecr-repo>.dkr.ecr.us-east-2.amazonaws.com
```

## Pull the Docker image
```bash
docker pull <your-cardinal-ecr-repo>.dkr.ecr.us-east-2.amazonaws.com/cardinal:latest
```

## Network Requirements

Cardinal-managed keys (default) → allow outbound HTTPS to
https://api.trycardinal.ai.

AWS Bedrock integration → inference stays inside your AWS account,
no Cardinal API whitelisting required.
